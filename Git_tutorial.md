# Памятка по работе с Git
## Основные команды
* git init
* git add
* git log
* git status
## Работа с ветками
* Создание веток
* Слияние веток
_**git merge**_ *(название ветки)* - Команда для слияния веток. Вызывается **из** той ветки, в которую нужно влить *(название ветки)*
* Разрешение конфликтов
